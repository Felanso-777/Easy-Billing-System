# 🧾 Easy Stock & Billing System

## 📌 About

This is a simple web application to manage shop products and perform billing.

It allows shop owners to:

* Add and manage products
* Create bills easily
* Calculate total in real-time

---

## 🚀 Features

* ➕ Add products (name, price, quantity)
* 📋 View all products
* ✏️ Update product details
* ❌ Delete products
* 🧮 Billing system:

  * Add items to bill
  * Increase / decrease quantity
  * Remove items
  * Clear full bill
  * Automatic total calculation

---

## 🛠️ Technologies Used

* Python (Flask)
* SQLite
* HTML
* Bootstrap
* JavaScript

---

## ▶️ How to Run

1. Install Flask:

   ```
   pip install flask
   ```

2. Run the application:

   ```
   python app.py
   ```

3. Open in browser:

   ```
   http://127.0.0.1:5000
   ```
